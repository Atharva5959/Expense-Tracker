# Personal Finance Tracker

A full-stack Personal Finance Tracker built using FastAPI, Supabase, JWT Authentication, HTML, CSS and JavaScript.

## Features

* User Registration
* User Login
* JWT Authentication
* Category Management
* Expense Management
* Budget Management
* Dashboard Analytics
* Expense Distribution Pie Chart
* Recent Expenses Tracking
* Budget Monitoring
* Responsive User Interface

## Technologies Used

### Backend

* Python
* FastAPI
* JWT Authentication
* Supabase

### Frontend

* HTML
* CSS
* JavaScript
* Chart.js
